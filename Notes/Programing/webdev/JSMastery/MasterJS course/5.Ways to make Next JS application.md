###### The file structure is ![[Pasted image 20260407193110.png]]
 - tsconfig.json => it say what is to be type checked and what is not 
 - README.md => simple markdown file
 - postcssconfig.mjs => contains css rules (like tailwind)![[Pasted image 20260407193254.png|286]]
 - package.json => contains all the dependencies and scripts(dev,build,start, lint) of the project
 - package-lock.json => it look the version of dependencies and sub-dependencies for exactness
 - next-config.ts => configure next js 
 - next-env.d.ts =>(not to be edited) for next js
 - eslint.config.mjs => it configure eslint 
 - .gitignore => has .env
 - public => has all static assets
 - app => our main application (most important)
