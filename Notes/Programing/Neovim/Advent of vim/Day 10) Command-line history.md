## 📘 Vim Command-line Window

### Command History (Ex Commands)

- In command mode (`:`), use **Up / Down Arrow** to cycle through previous commands
    
- `q:` → Open the **command-line window** with full command history  
    You can move the cursor, edit, and press `Enter` to re-run a command
    
- From command mode, `Ctrl + f` → Opens the same command-line history window
    

---

### Search History

- During forward search (`/`), use **Up / Down Arrow** to browse previous searches
    
- `q/` → Open **search history window** for forward searches
    
- From search mode (`/`), `Ctrl + f` → Opens the same search history window
    

---

### Forward vs Backward Search

- `/` → Search **forward** from the cursor
    
- `?` → Search **backward** from the cursor  
    (both maintain their own searchable history)
    

---

### Exiting the History Window

- `Ctrl + c` → Exit the command/search history panel
    

---

### Quick Memory Tip

- `q:` → command history
    
- `q/` → search history
    
- `Ctrl + f` → history from inside `:` or `/`
