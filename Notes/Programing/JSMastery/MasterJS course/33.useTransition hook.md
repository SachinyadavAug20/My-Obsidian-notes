[useTransition – React](https://react.dev/reference/react/useTransition)
###### It used to render part of UI in the background
###### Instead of managing loading states manually(if form submitted or not) you can use this hook
Declare as
```TS
  const [isPending,startTransition]=useTransition();
```
Use as in a question submission form
```TS
const handleCreateQuestion = async (
    data: z.infer<typeof AskQuestionSchema>,
  ) => {
    startTransition(async () => {
      const result = await createQuestion(data);
      if (result.success) {
        toast.success("Success", {
          description: "Question created successfully",
        });
        if (result.data) router.push(`${ROUTES.QUESTIONS}/${result.data?._id}`);
        else {
          toast(`Error ${result.status}`, {
            description: result?.error?.message || "Something went wrong",
          });
        }
      }
      console.log(data);
    });
  };
  ...
  // submit btn
  <Button
            type="submit"
            form="form-rhf-input"
            disabled={isPending}
            className="primary-gradient !text-light-900 w-fit px-5 py-1"
          >
            {isPending ? (
              <>
                <ReloadIcon className="mr-r-2 size-4 animate-spin" />
                <span>Submitting</span>
              </>
            ) : (
              <> Ask a question</>
            )}
</Button>
```
###### Advantages of using this instead of useState
- It is not blocking UI stays responsive
