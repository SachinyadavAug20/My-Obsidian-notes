## No-need-sequential request
###### as request block rendering which leads to bad SEO. example in right-side-bar => display hot-question and popular-tags
```TSX
  const { success, data: hotQuestion, error } = await getHotQuestion();
  const {success:tagSuccess,data:tags,error:tagError}=await getPopularTags()
```
###### This is sequential request block each other and render. can do in parallel
```TSX
  const [{  success, data: hotQuestion, error },{ success:tagSuccess,data:tags,error:tagError}]=await Promise.all([getHotQuestion(),getPopularTags()])
```
