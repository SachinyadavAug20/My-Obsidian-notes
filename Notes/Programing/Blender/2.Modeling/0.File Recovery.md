Before diving into Blender and creating your art, we need to make sure how to recover our files. Here are different ways to prevent data loss when working in and with Blender:

- Using [**Recover Last Session**](https://docs.blender.org/manual/en/latest/files/blend/open_save.html#last-session).
- Using [**Auto Saves**](https://docs.blender.org/manual/en/latest/files/blend/open_save.html#auto-save).
- Using a personal server/machine with data recovery enabled.
- Using a Cloud service to add an extra layer of safety.
- Using an SVN repository (which is a server).

---
##### [Recover Last Session](https://studio.blender.org/training/blender-fundamentals-45-lts/blender_4-5_lts_file-recovery/#recover-last-session)

Will restore the last session (without that file being saved) you opened and might be able to recover your data.

This can be accessed by going to **File > Recover > Last Session**.
