###### there are short-cuts for each mode in view-port

| key | fn                                                          |
| --- | ----------------------------------------------------------- |
| q   | select mode                                                 |
| w   | move mode                                                   |
| e   | rotate                                                      |
| r   | scale                                                       |
| t   | toggle between global and local space(axis)                 |
| g   | palm mode(move view-port) / also can do by middle mouse btn |
| m   | ruler to measure distance                                   |
